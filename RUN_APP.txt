RUN STREAMLIT APP (EDGAR)

FASTEST OPTION IF YOU ARE USING WSL / BASH
- Open RStudio Terminal or any Bash terminal
- Run:

cd /mnt/c/Users/DELL/Desktop/MartinGit/edgar
.venv/bin/python -m streamlit run app.py

If `.venv` does not exist yet, create it once:

python3 -m venv .venv
.venv/bin/python -m pip install --upgrade pip
.venv/bin/python -m pip install -r requirements.txt

If `python3 -m venv .venv` fails with an `ensurepip` / `python3-venv` error, run first:

sudo apt update
sudo apt install -y python3-venv python3-pip

FASTEST OPTION IF YOU ARE USING WINDOWS POWERSHELL
- Open PowerShell in the repo folder
- Run:

cd "C:\Users\DELL\Desktop\MartinGit\edgar"
powershell -ExecutionPolicy Bypass -File .\run_preview.ps1

Useful variants:

powershell -ExecutionPolicy Bypass -File .\run_preview.ps1 -RefreshDeps
powershell -ExecutionPolicy Bypass -File .\run_preview.ps1 -Port 8502

NOTES
- The repo's current `.venv` is Linux/WSL-only. The PowerShell helper now prefers a real Windows Python when available, but automatically falls back to the existing WSL environment if PowerShell only sees the Microsoft Store alias.
- Do NOT press Ctrl + C unless you want to stop the app.
- Open the preview in your browser at:
  http://localhost:8501
- Streamlit Community Cloud is for the deployed app after commit/push; local preview is the fastest way to inspect edits before that.
